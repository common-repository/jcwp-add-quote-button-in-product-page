A plugin that adds a WhatsApp quote button on the WooCommerce product and shop page.

== Description ==

The "Add Quote" button for WhatsApp in WooCommerce is a feature that allows customers to request quotes directly through the e-commerce platform using WhatsApp. This facilitates communication between the customer and the seller and allows for quick and efficient response to questions and quote requests. Additionally, by being integrated with WooCommerce, the "Add Quote" button seamlessly integrates into your online store and can be customized to fit your needs.


== Installation ==

= Minimum Requirements =

* PHP 7.2 or greater is recommended
* MySQL 5.6 or greater is recommended

1. Upload `quote-button-by-whatsapp` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the settings in: Settings/Quote Button by WhatsApp
